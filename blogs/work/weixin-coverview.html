微信小程序中使用cover-view组件遇到的问题 ​

uni-app、web-view组件、cover-view组件无效 ​

  • 坏的方式会导致部分安卓机型（小米、vivo、一加、荣耀等）在使用cover-view组件失效，ios、华为正常

bad

vue
<template>
	<view>
		<web-view src="https://www.baidu.com"></web-view>
		<cover-view class="cover-block">
			<cover-view>文本</cover-view>
		</cover-view>
	</view>
</template>
<style>
.cover-block{
    position:fixed;
    left:0;
    right:0;
    top:0;
}
</style>

vue
<template>
	<view>
		<web-view src="https://www.baidu.com">
            <cover-view class="cover-block">
                <cover-view>文本</cover-view>
            </cover-view>
        </web-view>
	</view>
</template>
<style>
.cover-block{
    position:fixed;
    left:0;
    right:0;
    top:0;
}
</style>
  • 给cover-view组件添加z-index

good

vue
<template>
	<view>
		<web-view src="https://www.baidu.com"></web-view>
		<cover-view class="cover-block">
			<cover-view>文本</cover-view>
		</cover-view>
	</view>
</template>
<style>
.cover-block{
    position:fixed;
    left:0;
    right:0;
    top:0;
    z-index:1;
}
</style>

vue
<template>
	<view>
		<web-view src="https://www.baidu.com">
            <cover-view class="cover-block">
                <cover-view>文本</cover-view>
            </cover-view>
        </web-view>
	</view>
</template>
<style>
.cover-block{
    position:fixed;
    left:0;
    right:0;
    top:0;
    z-index:1;
}
</style>
