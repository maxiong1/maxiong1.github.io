css.modules ​

起因是less文件:export导出的变量通过import {primaryColor75} from '@/assets/less/module/xx.module.less'控制台报错，无法正常获取primaryColor75的值。

所以使用了<style module>,示例

vue
<template>
    <div :class="$style.xxx">
        11
    </div>
</template>
<script>
export default {
    name: 'component',
    create(){
        conosle.log(this.$style) 
    }
}
</script>
<style lang="less" module> 
    @import '~@assets/less/module/xx.module.less';
</style>
