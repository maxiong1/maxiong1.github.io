    产品需求是：接口每30秒返回多车经纬度，前端展示车的运动轨迹；车需要展示label；点击车弹窗展示基本信息；

    # 两个经纬度之间计算角度

    calcAngle(start,end) { 
    // lngLatToContainer 经纬度坐标转换像素坐标
    // Math.atan2(y,x) 返回弧度值
    // 2Π(弧度) = 360
    var p_start = this.map.lngLatToContainer(start),
        p_end = this.map.lngLatToContainer(end);
    var diff_x = p_end.x - p_start.x,
        diff_y = p_end.y - p_start.y;
    return 360*Math.atan2(diff_y, diff_x)/(2*Math.PI);
}

    # 添加了行走停止状态

    • 初始状态下会记录小车运动到第几个paths里面元素
    • 初始状态下的小车状态时停止，如果paths里面有经纬度才会运动
    • 判断上下两个经纬度是否一样，如果一样不插入到paths里面
    • 只有小车处于停止状态且存在下一个经纬度的时候才会执行moveTo
