# 回滚到某一个版本

  • git reset --hard/--soft commit_id
    • soft 保留历史记录
    • hard 不保留历史纪录
  • git push <branch> --force

fatal: The upstream branch of your current branch does not matchthe name of your current branch. 致命：当前分支的上游分支不匹配当前分支的名称。

  • git push -u origin <branch>
